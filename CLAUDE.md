# twill

TWiLL (Type-Woven, invariant-Led Layering) モノリポ。

## 構成

- bare repo: `~/projects/local-projects/twill/.bare`
- main worktree: `~/projects/local-projects/twill/main/`
- feature worktrees: `~/projects/local-projects/twill/worktrees/<branch>/`

## ディレクトリ構造

- `cli/` — TWiLL CLI エンジン（`twl` コマンド）
- `plugins/dev/` — 開発ワークフロープラグイン
- `plugins/session/` — セッション管理プラグイン
- `test-fixtures/` — テスト用フィクスチャ
