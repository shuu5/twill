## 受け入れ基準（Issue #114）

1. `.github/workflows/add-to-project.yml` が存在する
2. `.github/workflows/project-status-done.yml` が存在する
3. loom-plugin-dev で新規 Issue 作成 → Board に自動追加される（PAT 登録後）
4. loom-plugin-dev で Issue close → Board Status が Done に遷移する
5. Board に未登録の Issue を close した場合、workflow run が success (green) で完了する
6. PR description に PAT 作成手順 + 3リポ Secret 登録手順が記載されている
