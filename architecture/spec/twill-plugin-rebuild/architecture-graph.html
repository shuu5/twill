§3. 使い方

  • node click: 各 circle を click → 対応 spec file へ遷移 (<a xlink:href> による native browser navigation)
  • node hover: tooltip で file 概要表示 (<title>) + circle stroke 強調 + brightness 上昇 (CSS :hover)
  • category 区分: 7 column layout (A→G、左から右へ)、color で category 識別
  • edge 種別: 太線 (brand 色) = hub edge (5 hub file から放射)、細線 (gray) = within-category structural link
  • responsive: 画面幅 1400px 以下でも auto-scale (max-width: 1400px、aspect-ratio 保持)

§4. 維持方針

本 graph は 手動更新 (Phase J 時点)。spec file 追加 / 削除時は本 file の SVG block を edit する。将来 (Phase 2 以降) の自動生成 script (scripts/spec-graph-gen.py) は rebuild-plan.html J-5 (optional) で検討。
